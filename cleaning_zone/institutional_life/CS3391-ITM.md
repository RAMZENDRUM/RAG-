Description 

Multithreading in Java allows developers to execute 

multiple threads (smaller units of a program) 

concurrently within a single application. This enables 

efficient utilization of CPU resources, making Java 

applications more responsive and capable of handlin g

complex tasks. Java's multithreading capabilities are 

achieved through its Thread class and Runnable 

interface, allowing developers to create and manage 

threads easily. By carefully designing and 

synchronizing threads, Java developers can build 

responsiv e and efficient applications, making it a 

crucial aspect of modern Java programming. 

Inheritance in Java is a fundamental 

object -oriented programming concept that allows 

classes to inherit attributes and methods from other 

classes. I t establishes a parent -child relationship, 

where a subclass (child) can inherit properties and 

behaviors from a superclass (parent). This promotes 

code reusability and the creation of hierarchical class 

structures. Subclasses can extend the functionality o f

their parent classes by adding new methods or 

overriding existing ones. Java's inheritance 

mechanism is a powerful tool for building modular 

and extensible code, enabling developers to design 

more efficient and organized applications.
